
# Team Task Manager

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- JWT Authentication

## Backend Setup
cd backend
npm install
npm run dev

## Frontend Setup
cd frontend
npm install
npm run dev

## Deployment
### Railway Backend
1. Push backend to GitHub
2. Create Railway project
3. Add environment variables
4. Deploy

### Vercel Frontend
1. Push frontend to GitHub
2. Deploy with Vercel

## Features
- Authentication
- Projects
- Tasks
- Dashboard
- Role-based Access
